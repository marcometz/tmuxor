DESCRIPTION
-----------
TMUXor — your tmux Claude Code fleet, hands-free on your glasses.

TMUXor is a heads-up conductor for developers who run many Claude Code sessions
at once. It brings the sessions running in your tmux panes onto your Even G2
glasses so you can stay in flow without reaching for the laptop.

• Organized fleet — tmux windows become project tags; each pane is one session,
  listed by tag so you can find things fast.
• Real conversations — read the actual prompts and replies (not raw terminal
  noise). Reopen a session right where you left off — or jump straight to a new
  question if one arrived while you were away.
• Voice replies — tap, speak, review the transcription (with its cost), send.
  Approve interactive menu choices with a tap.
• Start new sessions — pick a project tag, speak a folder, and TMUXor opens a
  Claude session there.
• Private by design — talks only to your own backend on your own machine, over
  Tailscale, behind a token you set. Nothing is sent to the app developer.

Requires a one-time backend setup on your computer (tmux + Claude Code +
Tailscale). Setup guide included.

PERMISSIONS (justifications)
----------------------------
network — Connects to your own TMUXor backend over Tailscale to read and drive
your tmux sessions. (Whitelisted to Tailscale hosts; the app talks only to the
backend URL you enter in Setup.)

g2-microphone — Captures glasses microphone audio for voice replies, transcribed
by your configured backend.


PRIVACY POLICY
--------------
TMUXor collects no personal data and sends nothing to its developer. It connects
only to the backend URL you configure — running on your own device, reached over
your private Tailscale network — using an access token you provide. Microphone
audio you record is sent to your configured backend, which transcribes it (by
default via OpenAI's Whisper API using your own API key) and does not retain it.
Your backend URL, access token, and preferences are stored locally on your phone
only. The app contains no analytics, ads, or third-party tracking. You are
responsible for securing your backend and API keys.
