TMUXor — see and continue your tmux + Claude Code sessions from your Even G2 glasses.

Run a fleet of Claude Code sessions on your computer? TMUXor puts them on your glasses so your work stays trackable, hands-free — over your own private Tailscale network. Nothing is sent to the app developer.

• Tagged fleet, sorted by what needs you — each tmux window is a project tag.
• Read the real conversation; double-tap jumps back to the latest prompt.
• Reply by voice (tap, speak, send) — or type on your phone if you have no OpenAI key.
• Approve safely — read the full command on the glasses before you pick Yes/No.
• Start a session — pick a tag, name a folder, and Claude opens there.

Controls: tap = do · double-tap = back · swipe = move/scroll.

Setup (one time): full guide at github.com/marcometz/tmuxor — run install.sh on your computer, then paste the printed config code into the app. Needs tmux or Herdr + Tailscale.
